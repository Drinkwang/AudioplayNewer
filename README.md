# AudioplayNewer
一个播放声音的对接讯飞sdk的垃圾项目bate1.0
自行取用，如有侵权，告知可删除
代码是讯飞的sdk，除了这块，其他逻辑遵循BSD开源协议，可商用
代码可能存在一些粗俗的命名，这些是因为当时情绪低落所致，没有别的含义，请勿介意。
可自行修改，贩卖，改进，但如有可能尽可能开源，感谢
BSD开源协议
